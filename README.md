# Mining Safety Simulation

A simple Python simulation of a layered mining safety system, including sensor integration, processing, and alerts.

## How to Run
- Requires Python 3.x.
- Install dependencies: `pip install numpy` (optional, for enhancements).
- Run: `python mining_sim.py`
- It simulates real-time data every 2 seconds. Press Ctrl+C to stop.

## Features
- Integration Layer: Fake thermal and camera sensors.
- Processing Layer: Risk assessment and alerts.
- Application Layer: Dashboard output.
